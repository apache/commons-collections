This package contains collection decorators that allow you to register to listen to changes in the underlying collection. A standard listener based event system is supplied, but alternatives can be added, see ModificationHandler.

Each decorator can be constructed using a static decorate() method on the ObservableXxx class. They can also be constructed from the XxxUtils class (where Xxx is the collection type).

List observableList = ObservableList.decorate(new ArrayList(), listener);
OR
List observableList = ListUtils.observableList(new ArrayList(), listener);
